Before you start: This 8-week plan assumes you already have a basic running base. If you‚Äôre brand-new to running, build up first (consistent easy runs for 6‚Äì8 weeks).

A good baseline for any level is being able to complete a 6‚Äì8 mile long run comfortably (run or run/walk) and running at least 3 days/week.

Level 1 (Finish-focused)
  • Running 3 days/week consistently
  • Longest recent run: ~6 miles (run/walk OK)
  • Comfortable easy effort (conversation pace)
  • Goal: finish strong and feel good afterward
Level 2 (Steady performance)
  • Running 3‚Äì4 days/week consistently
  • Longest recent run: ~8‚Äì9 miles
  • Can run continuously for ~60‚Äì90 minutes
  • Goal: steady pacing with a strong finish
Level 3 (Race-focused)
  • Running 4‚Äì5 days/week (including workouts)
  • Longest recent run: ~9‚Äì10 miles
  • Comfortable with tempo/threshold work
  • Goal: race the half marathon (not just finish)
